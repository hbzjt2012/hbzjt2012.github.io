Hexo+Next搭建个人博客

发表于 2019-06-04 | 分类于 Hexo | | 浏览 次
字数统计: 492 字 | 阅读时长 ≈ 1 分钟

缘由

  虽然目前的博客网站有很多，如CSDN、博客园等等，但是出于个人方便和某种心理作用，一直想搭建一个属于自己的博客网站，当然如果大家浏览国外网站比较多的话，很多外国朋友都有搭建自己的个人网站来记录学习或者生活点滴的习惯。
  但是奈何自己的学识有限，因此很长一段时间这个计划没有成形，前不久发现大家使用Hexo搭建博客比较方便，对于我这种入门小白，再合适不过了，使用Hexo加Next主题不用太长时间就可以搭建一个漂亮的个人博客网站出来，经过几天的折腾，现在博客网站已经有了雏形，后续会再进行不断的优化。

  从今天开始再此记录自己的学习记录，同时与大家分享自己的生活点滴，希望以后可以在这里找到美好的回忆！

博客搭建

  关于使用Hexo+Git+Next搭建博客的详细教程，已经有前人写的非常详细，在此列出个人参考的两个教程，并在此向原作者表示感谢！

后续会陆续将自己以前在各个网站发布的博客和文章迁移到此，并进行整理，开始自己的新的学习历程！🙂
